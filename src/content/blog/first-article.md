# Введение в Astro

Astro — это современный фреймворк для создания быстрых веб-сайтов с фокусом на контент.

![Пример изображения](/blog/images/example.png)

## Основные преимущества

- **Быстрая загрузка**: Отправляет минимум JavaScript клиенту
- **Простота**: Используйте привычные технологии
- **Гибкость**: Поддержка React, Vue, Svelte и других фреймворков

## Пример кода

```javascript
// Простой пример JavaScript
function greet(name) {
  return `Привет, ${name}!`;
}

console.log(greet('Мир'));
```

## Заключение

С Astro создавать блоги стало проще! Просто добавляйте Markdown файлы, и они автоматически превращаются в страницы.
